# vimrc読書会 要件

* チャットのページとしては常に同じがいい
* 一般的なテキストチャットの機能が必要
* チャンネルは NIP-28 (Public Chat) を使用する（kind:40 でチャンネル作成、kind:42 でメッセージ投稿）
* MUST
  * メッセージの投稿
  * メッセージのリアルタイム性
  * マークアップ
  * Permalink
  * ユーザーの区別
  * ログインしなくても外から誰でも見れる
  * アカウントの作成
  * アカウントのプロフィール設定(名前)
  * botが投稿できる/読める
* SHOULD
  * メッセージの編集
  * アカウント(鍵)のアイコン設定
* MAY
  * メッセージの削除
  * メッセージへのリアクション
  * メッセージへの返信
  * メッセージへの引用
* 不要
  * メッセージのリポスト
* 実装済み追加機能
  * GitHubファイルリンクのコードプレビュー（blob URLからソースを取得しハイライト表示、行範囲指定対応）
  * 当日の参加者リスト（左ペイン、AM5:00 JST切り替え）
  * ログイン状態のlocalStorage永続化
  * 鍵生成時の名前設定（kind:0投稿）
  * nsecのマスク表示・コピーボタン
  * メッセージのlocalStorageキャッシュとページング表示（50件ずつ無限スクロール）
  * バックグラウンドWebSocket同期（ページ遷移してもリレー接続維持）
  * 設定画面（キャッシュクリア、GitHubプレビュー切替）
  * メッセージ削除の確認ダイアログ（ゴミ箱アイコン + 確認モーダル）
  * 削除時のlocalStorage物理削除（リロード後も復活しない）
  * リレー接続状況パネル（ヘッダーのドットクリックで各リレーの状態表示）
  * Vim/ターミナル風ダークテーマ（常時ダーク、gruvbox系カラー、JetBrains Mono）
  * 投稿フォーム自動高さ調整（行数に応じてtextareaが拡縮）
  * Cloudflare Workersデプロイ対応
