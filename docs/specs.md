# vimrc読書会 要件

## 概要

vimrc読書会用のNostrチャットクライアント。NIP-28 (Public Chat) ベースで、単一チャンネルのテキストチャットを提供する。

- チャンネルは NIP-28 を使用（kind:40 でチャンネル作成、kind:42 でメッセージ投稿）
- 未ログインでも誰でも閲覧可能
- botからの投稿・読み取りに対応

## 機能要件

### 実装済み

| 分類 | 機能 |
|------|------|
| チャット | メッセージ投稿・リアルタイム受信 |
| チャット | Markdownレンダリング |
| チャット | Permalink（neventパラメータ） |
| チャット | メッセージ削除（kind:5、確認ダイアログ付き） |
| チャット | メッセージへのリアクション |
| チャット | メッセージキャッシュと無限スクロール |
| チャット | 投稿フォーム自動高さ調整 |
| チャット | OGPリンクプレビュー |
| チャット | GitHubファイルリンクのコードプレビュー（行範囲指定対応） |
| ユーザー | アカウント作成（鍵生成時に名前・アイコン設定） |
| ユーザー | アイコン設定（URL入力 / クリップボードから画像アップロード） |
| ユーザー | NIP-07ブラウザ拡張ログイン（ローディング表示付き） |
| ユーザー | nsecのマスク表示・コピーボタン |
| ユーザー | ログイン状態の永続化 |
| ユーザー | ログアウト確認ダイアログ |
| ユーザー | ユーザー表示名・アイコンの区別 |
| ユーザー | 当日の参加者リスト（AM5:00 JST切り替え） |
| インフラ | バックグラウンド同期（ページ遷移してもリレー接続維持） |
| インフラ | リレー接続状況パネル |
| インフラ | 画像アップロード（NIP-96/NIP-98対応、アップロード先変更可能） |
| インフラ | 設定画面（キャッシュクリア、GitHubプレビュー切替、画像アップロード先） |
| インフラ | Cloudflare Workersデプロイ対応 |
| UI | Vim/ターミナル風ダークテーマ（gruvbox系カラー、JetBrains Mono） |
