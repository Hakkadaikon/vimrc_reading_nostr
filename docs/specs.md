# vimrc読書会 要件

* チャットのページとしては常に同じがいい
* 一般的なテキストチャットの機能が必要
* チャンネルは NIP-28 (Public Chat) を使用する（kind:40 でチャンネル作成、kind:42 でメッセージ投稿）
* MUST
  * メッセージの投稿
  * メッセージのリアルタイム性
  * マークアップ
  * Permalink
  * ユーザーの区別
  * ログインしなくても外から誰でも見れる
  * アカウントの作成
  * アカウントのプロフィール設定(名前)
  * botが投稿できる/読める
* SHOULD
  * メッセージの編集
  * アカウント(鍵)のアイコン設定
* MAY
  * メッセージの削除
  * メッセージへのリアクション
  * メッセージへの返信
  * メッセージへの引用
* 不要
  * メッセージのリポスト
